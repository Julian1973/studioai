# Crystal Bears — LoRA Training-Image Prompts

20 individual image prompts per character (220 total). Each prompt generates ONE clean, single-character, on-model image; the 20 images per character become that character's LoRA training dataset.

## How to use
1. Generate each of a character's 20 prompts → 20 individual images (one character alone, no text/labels).
2. Drop them in `cb-seed/training/<Character>/`.
3. Train with `train_lora.py` (fal flux-lora-fast-training), captioned with the character's **trigger word** (below).
4. Once trained, the character renders identically every time — the drift cure.

## Notes for a strong dataset
- Keep the CHARACTER identical across all 20 — only angle / distance / pose / expression / light / background change.
- ~16 are on a clean neutral studio backdrop (helps the LoRA isolate the character) + ~4 in-world.
- Aim for ~1024px+ per image; tack-sharp face on the portrait shots.
- One character per image — never two, never text in frame.
