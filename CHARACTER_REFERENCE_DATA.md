# Crystal Bears — Character Reference Sheet Data

Everything needed to build a "Character Reference Sheet" (Buzzy-card style) for every character, externally, then bring the finished sheets back into the system. Source of truth = `cb-gen/config/characters.json` (do not invent — match this).

The MASTER turnaround for each character is named below (`turn4`) — when you build a sheet, the FRONT / 3-4 / SIDE / BACK on the "Main Identity + Scale" panel must match that turnaround exactly (identity is written in stone).

---

## PART 1 — THE SHARED TEMPLATE (identical on EVERY character's sheet)

These panels use the SAME labels for all 11 characters — only the character pictured changes. Reuse verbatim:

- **MAIN IDENTITY + SCALE SHEET** — Front view · 3/4 view · Side view · Back view, on a height ruler.
  - Ruler range: **bees 0'0"–2'0"**, **bears/cubs & guests 0'0"–6'0"**. Mark the character's exact height (below).
- **SILHOUETTE GUIDE** — Neutral stance silhouette + Profile silhouette.
- **EXPRESSION PROGRESSION (8):** Neutral · Curious · Worried · Surprised · Afraid · Sad · Determined · Relieved.
- **HEAD DETAIL SHEET:** 3/4 Headshot · Side Headshot · Top Angle · Low Angle · Diagonal Angle.
- **MICRO EXPRESSIONS (5):** Subtle Eye Tension · Slight Smirk · Lip Tension · Micro Fear · Controlled Breath.
- **NEUTRAL BASELINE + POSTURE VARIATION:** Relaxed (baseline) · Relaxed · Tense · Confident.
- **HAND / PAW GESTURES (4):** Relaxed Hand · Tense Fingers · Pointing · Subtle Gesture Near Face.

**Two range notes** (the labels stay the same; the *amplitude* differs):
- **Zenny** plays everything as MICRO — her "expressions" are tiny (a blink, a flat stare). Keep the 8 labels but render them small/deadpan.
- **Squeaky** (dolphin) has no hands; swap "Hand Gestures" for **Body/Tail poses** (leap · roll · snout-bump · distress-thrash) and read expression through the eyes + body, not a mouth.

Per-character header fields map like this: **Name · Alias = title · Role = pillar/function · Age · Personality · Core Theme = pillar+lesson · Speech Accent = cadence+voice**.
