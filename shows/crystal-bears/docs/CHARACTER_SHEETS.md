# Crystal Bears — Character Sheets (two-sheet set)

Each character has TWO sheets, built separately. Copy one fenced block at a time.
- **Sheet 1 — Identity & Scale:** the key six (header · palette · scale · silhouette+posture · wardrobe/special-traits · prop) around the turnaround. This is the canonical lock.
- **Sheet 2 — Expression & Pose:** expressions / head angles / micro / posture / gestures — the range (the LoRA + emotional-keyframe reference).

Shared rules: the Front/3-4/Side/Back **must match the named master turnaround** (identity is locked — don't invent). Ruler: **bees 0–2'**, **bears/cubs/guests 0–6'**, **Squeaky in water**. Keep the **yellow a muted honey tone** (per the turnarounds), not candy-bright.
